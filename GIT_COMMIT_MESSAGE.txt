Release v0.2.0 - Bible Reference Formatter

Major new feature: Bible reference parsing and formatting

Added:
- BibleReferenceFormatter class for parsing Bible references
- Support for 8 reference formats (single verse, ranges, multi-chapter, complex patterns, etc.)
- Cross-format compatibility (OSIS, USFX, Zefania)
- New data models: BibleReference, VerseRange, ReferenceFormatError
- Convenience method: get_verses_from_reference()
- 50 comprehensive tests with 70% code coverage
- New example: reference_formatter_example.py
- Updated documentation and README

Changed:
- Bumped version to 0.2.0

- Enhanced examples README

Security:

- Complexity limits to prevent abuse
- Proper error handling
